# .dotfiles

## 環境

- zsh
- vim >= 8.0

## 設定方法

1. 以下を実行

    ```shell
    $ git clone https://github.com/s14t284/.dotfiles.git ~/.dotfiles
    $ cd .dotfiles
    $ zsh setup.sh
    ```

1. iterm2 の Preferences から、「General」 -> 「Preferences」を選択し、`LOAD preferences from a custom folder or URL` にチェックを入れて `~/.dotfiles/iterm` を入力
